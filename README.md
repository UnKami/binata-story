# Binata Growth Story - Roadmap

This project is a single-page interactive roadmap for the Binata initiative, showcasing its growth and organizational structure.

## Features
- Interactive Timeline with drill-down views.
- Growth Chart using Chart.js.
- Interactive Organizational Graph using D3.js.
- RTL support (Hebrew).
- Responsive design with Tailwind CSS.

## How to Run

### Method 1: Open directly
Simply open [index.html](index.html) in any modern web browser.

### Method 2: Live Server (Recommended)
If you have VS Code, you can use the "Live Server" extension to run it.

### Method 3: Python Simple HTTP Server
If you have Python installed, you can run:
```bash
python -m http.server 8000
```
Then visit `http://localhost:8000` in your browser.
